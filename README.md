# Stellar-Classification-Dataset-


This competition is to identify a stellar as "GALAXY",  "QSO", or "STAR"

GALAXY（銀河）、STAR（恒星）、QSO（quasi stellar object:準星・クエーサー）は、宇宙を構成する全く異なるスケールと性質を持つ天体・構造

STAR（恒星）： 自ら光を放つガスの球体（例：太陽）。宇宙の基本単位。GALAXY（銀河）： 無数の恒星（数千万〜数兆個）やガスが重力で集まった巨大な集団。QSO（準星）： 超巨大ブラックホールが周囲の物質を飲み込む際に激しく輝く、非常に遠方にある活動銀河の中心核。
